# 🌸 CycleSync – PCOD-Aware Menstrual Health Tracking System

CycleSync is a full-stack MERN application designed to help users track menstrual cycles while providing **PCOD-focused insights and predictions**. Unlike traditional tracking apps, it combines **cycle data, symptoms, and lifestyle factors** to generate meaningful health insights.

---

# Features

-  User Authentication (JWT-based login/signup)
-  Cycle Tracking (period start & end)
-  Symptom Tracking (cramps, etc.)
-  Lifestyle Tracking (sleep, exercise)
-  Cycle Prediction (next period estimation)
-  Insight Engine (PCOD-related pattern detection)
-  Data Visualization (cycle trends using charts)
-  Responsive UI (React + Tailwind CSS)

---

# Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- Axios
- Chart.js

**Backend:**
- Node.js
- Express.js
- MongoDB
- Mongoose

**Authentication:**
- JSON Web Tokens (JWT)
